Identité Visuelle

Découvrez l'univers graphique de l'Ébénisterie Daniel Chamberland

Une identité forte qui reflète notre engagement envers l'excellence et le raffinement artisanal

Logo Principal

Notre logo représente l'union harmonieuse entre tradition et modernité, avec les initiales "DC" intégrées dans un cercle symbolisant la perfection artisanale.

Version Monochrome

  • Logo principal en noir sur fond blanc pour applications classiques
  • Parfait pour documents officiels, contrats et courrier d'affaires
  • Utilisé pour impression monochrome et fax
  • Maintenir un espace libre minimal de 15mm autour du logo

Variations de Couleurs

Le logo s'adapte harmonieusement aux fonds clairs et foncés avec des couleurs qui évoquent la chaleur du bois et l'élégance du travail artisanal.

Utilisation des Variations

  • Fond Clair (Gauche) : Dégradé brun/doré pour papeterie, site web sur fond beige
  • Fond Foncé (Droite) : Dégradé doré lumineux pour applications premium et supports sombres
  • Le contraste entre beige chaud et bleu foncé crée une identité distinctive
  • Toujours privilégier la lisibilité et le contraste approprié

Système Complet de Logos

Une famille complète de logos avec différentes tailles et variations pour s'adapter à tous les supports de communication.

Directives d'Application

  • Grande version : Pour en-têtes, signalétique extérieure (minimum 100mm)
  • Version intermédiaire : Pour cartes de visite, documents (minimum 30mm)
  • Petite version : Pour signatures email, applications web (minimum 20px)
  • Maintenir les proportions originales dans toutes les tailles
  • Ne jamais étirer, déformer ou modifier l'espacement des lettres

Applications Papeterie & Digital

Notre identité visuelle s'adapte harmonieusement aux supports imprimés et numériques, offrant une cohérence parfaite sur tous les points de contact.

Supports de Communication

  • Cartes de visite : Format standard 3.5" x 2", impression recto-verso avec finition mate
  • En-têtes de lettre : Format lettre (8.5" x 11"), logo en haut à droite
  • Enveloppes : Format #10, logo positionné en haut à gauche
  • Application mobile : Design responsive adapté aux écrans smartphones
  • Cohérence des couleurs beige/doré sur tous les supports

Papeterie Complète avec Logo Hexagonal

Une variation moderne avec logo hexagonal pour des applications contemporaines qui conservent l'élégance de notre identité visuelle.

Logo Hexagonal - Applications Spéciales

  • Design moderne : Variation hexagonale pour projets contemporains
  • Carte de visite premium : Fond sombre avec logo doré hexagonal
  • Papeterie complète : En-têtes, enveloppes et cartes coordonnées
  • Applications numériques : Interface mobile adaptée au logo hexagonal
  • Réservé aux applications modernes et supports premium

Applications Multisupports

Guide complet des applications sur supports numériques, imprimés et signalétique avec spécifications techniques pour chaque usage.

Applications Digitales

  • Site Web : Logo 200px minimum, format PNG/SVG
  • Email Signature : Logo 40px hauteur, format PNG optimisé
  • Réseaux Sociaux : Profil 400x400px, bannière 1200x630px
  • Applications Mobiles : Icône 180x180px pour iOS, 192x192px Android

Applications Imprimées

  • Cartes de visite : Logo 25mm minimum, 300 DPI
  • Papeterie : Logo 35mm, impression CMYK
  • Signalétique : Logo 200mm minimum, découpe vinyle
  • Véhicule : Logo 400mm, application adhésif professionnel

Typographie

Montserrat Bold & Montserrat SemiBold
Police principale pour le nom de l'entreprise, les titres et les en-têtes. Caractère moderne et élégant qui apporte sophistication et lisibilité optimale.

Usages : Titres, nom d'entreprise, menu de navigation, boutons d'appel à l'action

Source Sans 3
